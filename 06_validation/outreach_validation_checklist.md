# Outreach Validation Checklist (Spec-Kapitel 11)

- [ ] Wie viele Leads gesammelt?
- [ ] Wie viele qualifiziert?
- [ ] Wie viele angeschrieben?
- [ ] Wie viele Antworten?
- [ ] Welche Zielgruppe reagiert am besten?
- [ ] Welche E-Mail-Variante funktioniert besser?
- [ ] Welche Region funktioniert besser?

## Compliance-Gate (vor jedem Versand)
- [ ] Nur Geschäftsadressen, keine Privatpersonen?
- [ ] Bezug zur Zielgruppe hergestellt?
- [ ] Opt-out / Do-not-contact respektiert?
- [ ] Freigabe vorhanden (`APPROVED_TO_SEND_OUTREACH=true` oder ausdrücklich)?
- [ ] Versandlimit eingehalten?

## Content Validation
- [ ] Lokal relevant? · [ ] Klare Dienstleistung? · [ ] Klare Zielgruppe? · [ ] Brauchbarer CTA? · [ ] Intern verlinkt? · [ ] Menschlich geschrieben? · [ ] Kein KI-Müll?
